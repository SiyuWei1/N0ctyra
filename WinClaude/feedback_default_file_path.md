创建文件时默认保存到 OneDrive > SIyu W。

在 Mac 上路径为：`~/Library/CloudStorage/OneDrive-ThePennsylvaniaStateUniversity/SIyu W/`
在 Windows 上路径为：`C:\Users\Harrs\OneDrive - The Pennsylvania State University\SIyu W\`

**Why:** 用户将 SIyu W 文件夹从 iCloud 迁移到了 PSU OneDrive，方便在 Mac 和 Windows 之间同步。
**How to apply:** 除非用户指定其他路径，所有新建文件都放到 OneDrive 的 SIyu W 目录下。
