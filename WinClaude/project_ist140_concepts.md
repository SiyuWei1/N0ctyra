IST 140 (Spring 2026, Friedenberg) 已学内容：

**基础语法**
- while 循环、for 循环、嵌套循环
- if/else/elif 语句
- input()、print()、f-string
- print(..., end="")  不换行打印
- 基本变量、比较运算符、布尔运算（and/or/not）
- 布尔变量作 flag（found = True/False）

**类型转换**
- int()、float()、str() 类型转换

**字符串操作**
- 字符串方法：isupper()、islower()、isdigit()
- 字符串索引：string[i]
- 字符串遍历：for char in string
- 字符串拼接：str1 + str2
- 字符串重复：string * n
- len() 获取长度

**函数**
- def 定义函数、return 返回值
- 多个 return 语句
- 函数参数（单个、多个）
- 函数调用另一个函数
- from module import name 导入（如 from math import pi）

**数学**
- from math import pi

**列表**
- 列表创建、索引访问
- 列表遍历：for item in list
- random 模块（zyBooks 4.16）
