## Core Themes (贯穿全课的核心主题)

1. **社会脆弱性是灾害放大器** — 自然灾害+社会不平等=人为灾难
2. **了解风险是安全的第一步** — 知道你住的地方面临什么威胁
3. **Watch = 准备; Warning = 行动** — 两者区别攸关生死
4. **不要比较风暴** — 每次天气事件都是独一无二的
5. **气候变化正在改变所有灾害的风险格局**
6. **有效的灾害管理需要减灾+准备+响应+恢复四阶段持续循环**
