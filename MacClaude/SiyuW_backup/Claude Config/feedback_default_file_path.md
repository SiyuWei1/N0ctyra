---
name: 文件默认保存路径
description: 用户要求创建的文件默认保存到 OneDrive Siyu W 文件夹
type: feedback
---

创建文件时默认保存到 `~/Library/CloudStorage/OneDrive-ThePennsylvaniaStateUniversity/SIyu W/`。

**Why:** 用户将 Siyu W 文件夹从 iCloud 迁移到了 PSU OneDrive，方便在 Mac 和 Windows 之间同步。
**How to apply:** 除非用户指定其他路径，所有新建文件都放到 OneDrive 的 SIyu W 目录下。
